Hướng Dẫn Tự Động Hóa Auto-Sync Token

Phương pháp này chạy trực tiếp lệnh chặn trong Console F12 của trang game. Chắc chắn thành công, không lo bị security chặn:

  1. Mở trang game EE88, nhấn phím F12 và chọn thẻ Console.
  2. Copy toàn bộ mã bên dưới, dán vào ô nhập lệnh ở Console và nhấn Enter.
  3. Bấm click chuyển đổi trò chơi trên Menu của game (ví dụ: đổi từ 75s sang 45s, hoặc ngược lại). Lệnh sẽ tự động bắt lấy token mới khi game mở lại kết nối.

Phương pháp này sử dụng Bookmark (Trình đánh dấu) của trình duyệt. Cực kỳ nhanh, không cần cài extension, hoạt động trên mọi trình duyệt:

  1. Nhấn Ctrl + Shift + B để hiện thanh Bookmark trình duyệt lên (nếu chưa hiện).
  2. Chuột phải vào thanh Bookmark -> Chọn Thêm trang (Add page).
  3. Chuột trái vào ô URL và dán toàn bộ bookmarklet mã bên dưới. Lưu.
  4. Mở game và click bookmark đó, sau đó chuyển game để kích hoạt sync.

Nếu muốn tự động hóa hoàn toàn không cần click mỗi lần vào game, hay cài extension Tampermonkey và add script sau:

  1. Cài đặt extension Tampermonkey trên Cửa hàng Chrome/Firefox.
  2. Click icon Tampermonkey -> Chọn Create new script (Tạo script mới).
  3. Xóa hết code mặc định và copy toàn bộ script dưới đây dán vào. Ấn Ctrl + S để lưu.
  4. Truy cập trang game, nó sẽ tự động đồng bộ token về Dashboard mỗi khi game chạy lại.
